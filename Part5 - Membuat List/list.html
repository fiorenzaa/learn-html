Membuat List di HTML

Membuat List Terurut Menggunakan OL (Ordered List),
Contoh:

Prioritas Hidup

  1. Agama
  2. Akhlak dan Moral
  3. Ilmu Pengetahuan
  4. Kehidupan Sosial
  5. Penampilan

Defaultnya 1, 2, 3,
contoh yg lain:

Prioritas Hidup

  1. Agama
  2. Akhlak dan Moral
  3. Ilmu Pengetahuan
  4. Kehidupan Sosial
  5. Penampilan

Membuat List Tidak Terurut Menggunakan UL (Unordered List),
Contoh:

Things you have to be grateful for have them

  • Health
  • Keluarga
  • Hidup Berkecukupan
  • Circle yang tidak negatif
  • Masih dikasih waktu

Defaultnya bullets(disc),
contoh yang lain:

Things you have to be grateful for have them

  • Health
  • Keluarga
  • Hidup Berkecukupan
  • Circle yang tidak negatif
  • Masih dikasih waktu

Membuat Definition List menggunakan DL (Definition List),
contoh:

HTTP
Hypertext Transfer Protocol
HTTPS
Hypertext Transfer Protocol Secure

Contoh Definition Term nya diberi penekanan (bisa disetting di CSS sih)

HTML
Hypertext Markup Language
CSS
Cascading Style Sheet

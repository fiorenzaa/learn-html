Membuat Table di HTML

Tag 'table' di HTML punya banyak atribut, salah satunya:
  1. border
  2. cellspacing
  3. cellpadding
  4. rowspan
  5. colspan

*atribut lainnya bisa dilihat di www.w3schools.com

Membuat tabel dengan hanya menggunakan atribut border="1"

Baris 1,1 Baris 1,2 Baris 1,3
Baris 2,1 Baris 2,2 Baris 2,3
Baris 3,1 Baris 3,2 baris 3,3

Membuat tabel dengan menggunakan atribut border dan cellspacing="0"

Baris 1,1 Baris 1,2 Baris 1,3
Baris 2,1 Baris 2,2 Baris 2,3
Baris 3,1 Baris 3,2 Baris 3,3

Membuat table dengan Header

Header 1,1 Header 1,2 Header 1,3
Header 2,1 Baris 2,2 Baris 2,3
Header 3,1 Baris 3,2 Baris 3,3

Membuat tabel dengan menggunakan atribut border, cellspacing, dan cellpadding="10"

Baris 1,1 Baris 1,2 Baris 1,3
Baris 2,1 Baris 2,2 Baris 2,3
Baris 3,1 Baris 3,2 Baris 3,3
